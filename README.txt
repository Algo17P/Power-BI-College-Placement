
College Placement & Student Outcome Analytics – Power BI Project

Files Included:
1. College_Placement_Data.xlsx – Dataset
2. README.txt – Project instructions

Steps:
1. Open Power BI Desktop
2. Load College_Placement_Data.xlsx
3. Create KPIs:
   - Total Students
   - Placed Students
   - Placement Percentage
   - Average Package
   - Highest Package
4. Build visuals as per dashboard layout

This project is ideal for portfolios, internships, and interviews.
